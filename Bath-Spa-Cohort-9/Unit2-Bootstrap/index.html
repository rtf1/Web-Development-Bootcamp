I Dislike Bootstrap!

I personally would prefer to avoid using Bootstrap as it is adds considerable "bloat" to your css code and makes the webpage load slower e.g. 155kb for the base bootstrap file before you have even added your own code!

While bootstrap can be used to create front end components quickly, no employer looking for a junior front-end web developer would be impressed to find you using this in your portfolio (unless you are going for a full-stack position and have constructed a considerable server-side component to your project). Additionally, many apps or websites using Bootstrap end up looking very similar and this phenomenon can induce what is known as "Bootstrap Fatigue."
